# Food Review Project

A Django web application that allows users to review restaurants and manage food-related reviews.

## Features
- User registration/login
- Admin and regular user roles
- Restaurant CRUD operations
- Review CRUD operations
- Review moderation
- SQLite database integration
- API endpoints for restaurant/review data

## Tech Stack
- Python
- Django
- SQLite
- HTML/CSS
- Bootstrap
